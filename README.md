Infobip Bridge
==============

Provides Infobip integration for Symfony Mailer.

Configuration examples:

```dotenv
# API
MAILER_DSN=infobip+api://KEY@BASE_URL
# SMTP
MAILER_DSN=infobip+smtp://KEY@default
```
